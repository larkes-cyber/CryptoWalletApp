# SniWalletApp
Ton wallet app with all basic functions

Functions:
1. Create/import wallet
2. View transactions
3. Send transaction
4. Receive wallet
5. QR code scanner

Tech: 
Ton sdk, Kotlin, Jetpack Compose, Dagger-Hilt, MVVM
